# Atividade pós-aula

Desenvolva uma aplicação com frontend/cliente/consumidor e backend/servidor/produtor separados e integrados com REST

## Como rodar

Suba a aplicação backend com
`rails s -p 3000`

e então suba a aplicação frontend com
`rails s -p 3001`

Acesse http://localhost:3001/quoter/index e atualize a página para ver a frase mudar a cada atualização.
